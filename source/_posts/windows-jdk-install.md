# 配置windows环境变量：

![](windows-jdk-install/6.png)

![](windows-jdk-install/7.png)

![](windows-jdk-install/8.png)

```
1.
系统变量→寻找 Path 变量→编辑
在变量值最后输入 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

2.
系统变量→新建 JAVA_HOME 变量 。
变量值填写jdk的安装目录（本人是 E:\Java\jdk1.7.0)
```
