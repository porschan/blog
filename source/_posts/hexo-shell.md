## [hexo官网](https://hexo.io/zh-cn/docs/commands.html)

### 生成静态文件。

```
$ hexo generate
```

或

```
$ hexo g
```

### 启动服务器。默认情况下，访问网址为： http://localhost:4000/。

```
$ hexo server
```

或

```
$ hexo s
```

### 清除缓存文件 (db.json) 和已生成的静态文件 (public)。

```
$ hexo clean
```
