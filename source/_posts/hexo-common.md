### new ###

创建新的文章。	

```

	$ hexo new "hexo-article"

```

### generate ###

更新或生成文件页面内容。

```

	$ hexo generate

	or

	$ hexo g


```

### clean ###

清理所有文件信息。

```

	$ hexo clean

```

### server ###

开启hexo服务，默认localhost:4000访问

```

	$ hexo server

	or

	$ hexo s

```
